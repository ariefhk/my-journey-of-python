# My Journey of Python

A personal learning repository for exploring Python — from language basics to AI, databases, and web frameworks. Each topic is organized by Python version so examples can be tested and compared across releases.

## Structure

Each folder below is split into `3.10/` and `3.11/` subdirectories to hold version-specific code and notes.

| Folder                           | Topic                                           |
| -------------------------------- | ----------------------------------------------- |
| [`ai/`](ai/)                     | AI / machine learning experiments               |
| [`basic-python/`](basic-python/) | Core language fundamentals and exercises        |
| [`database/`](database/)         | Database access, ORMs, and queries              |
| [`fastapi/`](fastapi/)           | FastAPI web framework examples                  |
| [`starter/`](starter/)           | Project starters / boilerplate                  |
| [`tooling/`](tooling/)           | Dev tooling, linters, formatters, CLI utilities |
| [`utility/`](utility/)           | Reusable helper scripts and utilities           |
